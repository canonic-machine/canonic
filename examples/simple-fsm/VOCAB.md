## FSM-Specific Terms

### complete sentence
Text unit starting with capital letter and ending with period. Minimum requirement for readable content.

### draft state
Initial state where content may be incomplete or unformatted. Entry point for new content.

### published state
Terminal state where content has passed all quality gates. Represents validated, accepted output.

### quality gate
Validation check that must pass before state transition. Examples: capitalization check, spelling check, length requirement.

### review state
Intermediate state where content is complete but undergoing validation before publication.

### spelling error
Word not recognized by basic dictionary check. Simplified validation demonstrating semantic constraints.

### transition validation
Constraint checking performed when attempting state transition. Pass enables transition, fail triggers backflow.
